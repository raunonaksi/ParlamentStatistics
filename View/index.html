Seminar on Business Intelligence!

We wondered how Estonia parlament members are attending and voting during the official sessions. First we contacted the goverment for getting access to the API, however there wasn't any, so we scraped data from www.riigikogu.ee.

Dataset

Data was scraped from all the meetings, which were held between 30-03-2015 (First session of the new parlament) and to 23-05-2016. There are information about 120 members in the datadase. The maximum number of parlament members is 101, but due to the fact that some of the members reassigned. This means that these members where replaced. During that period there were 772 different votings.

Used frameworks:

  • Scrapy - Web scraping
  • Node.js, Express - Back-end
  • SQLlite - Database
  • HTML, Bootstrap, CSS - Front-end
  • D3 - Graphs
